Problem: Imagine you have the following table to paint with some colors. Can you accomplish this task with a minimum of these colors? How many colors?

Usage: You can choose a color in the select box below. You can click over the spots' table to color them with the picked color. If you want to clear a color already painted, you need to ensure that color is selected in select box and just click over the spot where it was painted.
